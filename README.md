# GitHub Repo Template

This repository serves as a template for creating new repositories with the same directory structure, branches, and files.

The following will be included in the `catalog-info.yaml` file for Velo and will need to be populated appropriately.
``` yaml
Capability: Enter capability here
CMDB Name: Enter CMDB system name here
LongCode: Enter LongCode here
```
## Usage
To create a new repository using this template, follow the
[GitHub Create a Repository](https://sportsbet.atlassian.net/wiki/spaces/CF/pages/11541610870)
